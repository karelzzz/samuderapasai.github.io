Tentang Penulis

Perkenalan tentang pembuat website

Fransiskus Karel Satria Tristano

Kelas : 10.3

Tujuan Website

Website ini dibuat sebagai bagian dari penilaian kolaboratif antara mata pelajaran Sejarah, Bahasa Indonesia, dan Informatika. Selain itu, website ini juga ditujukan bagi siapa saja yang ingin mempelajari secara lebih mendalam mengenai sejarah Kerajaan Samudera Pasai. Dengan menghadirkan informasi yang cukup lengkap serta akurat, diharapkan website ini dapat menjadi sumber belajar yang bermanfaat serta memperkaya wawasan para pelajar dan masyarakat umum tentang warisan sejarah Islam pertama di Indonesia.

Pengalaman

Sejujurnya, ketika saya menerima tugas kolaboratif ini, saya merasa cukup bingung karena harus menggunakan kode yang cukup kompleks. Terlebih lagi, saya belum menguasai banyak hal mengenai CSS, HTML, dan framework. Pada awalnya, saya kesulitan untuk memikirkan bagaimana cara membuat website ini agar terlihat menarik dan fungsional. Namun, setelah saya mengetahui bahwa pembuatan website ini diperbolehkan menggunakan bantuan artificial intelligence (AI), saya merasa sangat terbantu dalam mengembangkan website ini. Hal ini memotivasi saya untuk lebih mendalami dan memahami apa itu CSS, HTML, dan framework. Meskipun demikian, saya masih menghadapi satu tantangan utama, yaitu perbedaan antara materi yang diajarkan di sekolah dengan materi yang tersedia di internet. Perbedaan tersebut membuat saya harus mencari dan memverifikasi ulang informasi agar mendapatkan pemahaman yang akurat dan benar. Setelah melewati berbagai rintangan dan tantangan, akhirnya website ini dapat berkembang sesuai dengan harapan saya. Proses ini tidak hanya menambah wawasan saya tentang pembuatan website, tetapi juga mengajarkan saya untuk tetap berusaha dan belajar dalam menghadapi kesulitan.

Motivasi

Awalnya, saya membuat website ini hanya untuk menyelesaikan tugas dan mendapatkan nilai terbaik dari tiga mata pelajaran. Namun, selama prosesnya saya belajar banyak tentang pemrograman, meskipun masih dasar, seperti HTML, CSS, dan Framework. Bantuan teknologi seperti AI sangat membantu saya memahami pembuatan website. Pengalaman ini membuat saya tertarik untuk mendalami dunia coding, yang mungkin akan saya pilih sebagai jurusan kuliah nanti. Jadi, selain mendapatkan nilai, saya juga memperoleh wawasan baru yang bermanfaat untuk masa depan.
